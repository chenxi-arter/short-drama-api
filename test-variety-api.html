🔧 接口说明

数据结构说明

type字段含义:

  • 0 - 轮播图内容
  • 1001 - 搜索过滤器
  • -1 - 广告内容
  • 3 - 综艺视频内容

筛选器说明:

  • ids="0,0,0,0,0" - 最新上传
  • ids="1,0,0,0,0" - 人气高
  • ids="2,0,0,0,0" - 评分高
  • ids="3,0,0,0,0" - 最新更新
